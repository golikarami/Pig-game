A simple dice game built with HTML, CSS, and JavaScript.
Fast to play, easy to read, and perfect for practicing DOM manipulation and game logic.

Game Rules:

The game has two players.

Players take turns rolling a dice.

If the dice shows 1, the current score resets and the turn switches.

Any other number adds to the current score.

Players can Hold to save the current score to their total.

First player to reach 100 points wins. Glory is optional.


This project is mainly for:

Practicing JavaScript fundamentals

Understanding state management in small games

Getting comfortable with DOM events
